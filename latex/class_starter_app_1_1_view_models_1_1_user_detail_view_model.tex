\doxysubsection{Detailed Description}
View\+Model for managing user details including creation, editing, and role management. Handles both new user creation and existing user modification scenarios. 

This View\+Model provides comprehensive user management functionality including\+:
\begin{DoxyItemize}
\item \doxylink{class_starter_app_1_1_database_1_1_models_1_1_user}{User} creation with password validation
\item \doxylink{class_starter_app_1_1_database_1_1_models_1_1_user}{User} profile editing
\item \doxylink{class_starter_app_1_1_database_1_1_models_1_1_role}{Role} assignment and removal
\item \doxylink{class_starter_app_1_1_database_1_1_models_1_1_user}{User} deletion (soft delete)
\item Input validation and error handling 
\end{DoxyItemize}
