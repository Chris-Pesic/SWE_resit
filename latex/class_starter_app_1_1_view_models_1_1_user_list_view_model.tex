\doxysubsection{Detailed Description}
View\+Model for managing the user list display and interactions in the application. Provides functionality for loading, filtering, searching, and navigating users. Implements MVVM pattern with data binding support through INotify\+Property\+Changed. 

This View\+Model requires admin privileges to function properly. Non-\/admin users will be redirected to the main page when attempting to load users.

Key features\+:
\begin{DoxyItemize}
\item \doxylink{class_starter_app_1_1_database_1_1_models_1_1_user}{User} loading with role information
\item Real-\/time filtering by role and search text
\item Pull-\/to-\/refresh functionality
\item Navigation to user details and creation
\item Admin-\/only access control 
\end{DoxyItemize}
